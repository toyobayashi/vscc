* Node.js

* CMake

## Script option

### build.bat

* `Debug` / `Release`

* `Win32` / `x64`

* `dll` - Define CCPM_BUILD_DLL_\<unscoped_package_name\>

* `static` - Use `/MT & /MTd` instead of `/MD & /MDd`

* `test` - Define CCPM_BUILD_TEST=true

Example: `build.bat Win32 Debug static test`

### build.sh

* `Debug` / `Release`

* `dll` - Define CCPM_BUILD_DLL_\<unscoped_package_name\>

* `test` - Define CCPM_BUILD_TEST=true

Example: `./build.sh Debug test`
